# 📌 Aplicação Web com React

## 📖 Descrição do Projeto

Esta aplicação web foi desenvolvida utilizando React e tem como objetivo permitir que usuários realizem autenticação com uma conta Google e efetuem um cadastro simples utilizando informações obtidas automaticamente durante o login.

O sistema possui navegação entre páginas e armazenamento das informações básicas do usuário autenticado, como nome, e-mail e foto de perfil.

---

# 🚀 Funcionalidades

## 🏠 Tela Inicial (Home)

- Interface simples e intuitiva;
- Login com conta Google;
- Exibição das informações do usuário autenticado;
- Navegação entre as telas do sistema.

---

## 👥 Tela de Apresentação da Dupla

- Nome dos integrantes;
- Descrição breve de cada desenvolvedor;
- Interface organizada e agradável.

---

## 🧑‍🏫 Tela de Cadastro

- Formulário de cadastro;
- Pré-preenchimento automático dos dados obtidos no login;
- Geração de objeto JSON com os dados do usuário;
- Exibição do JSON na tela para validação.

---
